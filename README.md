# LoveMatch-AmirMahdis (فانتزی عاشقانه)

نسخه‌ای با طراحی فانتزی و انیمیشن‌های لطیف — آمادهٔ اجرا و دیپلوی روی Vercel.

## نکات تکمیلی
- فایل SVG کارتونی دو شخصیت در `src/icons/HeartCouple.jsx`.
- برای فونت Vazir، `Vazir.ttf` را در `public/fonts/Vazir.ttf` قرار دهید.
- اگر خواستید، می‌تونم نسخهٔ نهایی را به GitHub push کنم (به‌شرط اینکه دسترسی بهم بدی).

## اجرا
```bash
npm install
npm run dev
```
